## Featured Work
- **[SeqProfiler](https://github.com/Scrimas/SeqProfiler)**: A foundational bioinformatics pipeline written entirely from scratch in Python. It parses multi-FASTA files, identifies Open Reading Frames (ORFs) across both DNA strands, handles transcription and translation, and calculates key biochemical properties (Mass, Tm, pI, Extinction Coefficient) without relying on external dependencies (for calculations and logic at least) like Biopython.
- **[Wheat_Yield_Optimization_Analysis](https://github.com/Scrimas/Wheat_Yield_Optimization_Analysis)**: An agricultural data analysis investigating the combined effects of wheat genotype and fertilizer type on yield. Utilizes a Two-Way ANOVA pipeline, originally written in R and reproduced in Python.
- **[Poplar_Biomass_Prediction_Model](https://github.com/Scrimas/Poplar_Biomass_Prediction_Model)**: A biostatistical project estimating poplar tree biomass using non-destructive field measurements. Implements and validates a log-log regression model to establish allometric relationships, originally written in R and reproduced in Python.
- **[Sleep_Drug_Efficacy_Analysis](https://github.com/Scrimas/Sleep_Drug_Efficacy_Analysis)**: A comparative clinical data analysis of soporific drugs using paired samples. Automates the evaluation of normality assumptions to dynamically select between parametric and non-parametric tests (e.g., Wilcoxon Signed-Rank), originally written in R and reproduced in Python.
